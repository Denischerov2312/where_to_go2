# Куда пойти/Where to go
Проект представляет собой интерактивную карту с достопримечательностями и интересными местами.
Сайт показывает самые красивые и интересные места мира. На главной странице удобно просмотривать где находятся различные места, можно просмотреть их краткое описание, также увидеть картинки.
Имеются большие возможности для редактирования. Django админка позволяет легко редактировать, добавлять и изменять информация о достопримечательностях.

## Запуск на локальном сервере
 - Для запуска проекта вам понадобится Python. Инструкция по [установке Python](https://skillbox.ru/media/code/kak_zapustit_python/).
- Скачайте репозиторий с GitHub. Если у вас установлен git bash, можете использовать команду:
  ```bash
  git clone https://github.com/Denischerov2312/where_to_go2.git
  cd where_to_go2
  ```
- Создайте и активируйте виртуальное окружение(необязательно)
  ```
  python - m venv venv
  venv\Scripts\activate
  ```
- Затем используйте pip (или pip3, есть конфликт с Python2) для установки зависимостей:
  ```sh
  pip install -r requirements.txt
  ```
- Создайте файл базы данных и сразу примените все миграции командой `python3 manage.py migrate`
- Запустите сервер командой `python3 manage.py runserver`
